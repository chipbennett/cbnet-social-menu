Manage and display your social network profile links using a custom navigation menu.

== Description ==

This Plugin adds a custom Widget that allows you to add a list of social network profile links to any Theme, using a cusotm navigation menu. Using this 
method, you will keep your social profile links portable from Theme to Theme instead of having to re-enter them as Theme options each time you switch Themes.

== Installation ==

Manual installation:

1. Upload the `cbnet-social-menu` folder to the `/wp-content/plugins/` directory

Installation using "Add New Plugin"

1. From your Admin UI (Dashboard), use the menu to select Plugins -> Add New
2. Search for 'cbnet Social Menu'
3. Click the 'Install' button to open the plugin's repository listing
4. Click the 'Install' button

Activiation and Use

1. Activate the plugin through the 'Plugins' menu in WordPress
2. Create a new custom navigation menu, and name it, e.g. "Social". (The name isn't important, as long as you remember it!)
3. Add <strong>custom links</strong> to your "Social" menu, under the "Links" metabox, one for each social network profile you wish to include.
4. Save the menu, and assign it to the "Social Profile Links" Theme Location.
5. Go to Appearance -> Widgets, and add the "cbnet Social Profile Links" Widget to one of the Theme's dynamic sidebars
6. In the Widget confiuration, select the Font Icon to use

== Frequently Asked Questions ==

= What font icons are supported? =

Currently, the Plugin supports Genericons and Font Awesome

Most social networks are supported by both font icons, including Dribble, Dropbox, Facebook, Flickr, GitHub, Instagram, LinkedIn, Pinterest, Google+, Skype, Tumblr, Twitter, and YouTube.

Genericons adds support for Codepen, Digg, Path, Pocket, PollDaddy, Reddit, StumbleUpon, Vimeo, WordPress, and WordPress.com. To use one of these networks, you must select the Genericons font.

Font Awesome adds support for BitBucket, FourSquare, and StackExchange. To use one of these networks, you must select the Font Awesome font.
